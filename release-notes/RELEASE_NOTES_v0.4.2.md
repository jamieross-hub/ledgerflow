# LedgerFlow v0.4.2 发布说明

发布日期：2026-03-04

## 亮点更新

- 🌐 **国际化持续完善（zh/en）**
  - 新增并完善设置页、关于页的中英文文案（不仅限于左侧菜单）
  - 增加设置页语言切换选项，支持中文 / English 即时切换
  - 修复语言切换后中文偶发回退为英文的问题（language-only 解析与本地语言键稳定化）

- 🧠 **嵌入模型开始真正落地（Phase 1）**
  - 新增 OpenAI-Compatible Embedding 客户端（`/embeddings`）
  - 新增语义召回模块：
    - 交易数据向量化
    - 本地索引缓存（localStorage）
    - 余弦相似度检索 Top-K
  - AI 助手分析模式已接入“语义召回增强上下文”，并支持失败自动降级

- 🎨 **侧边栏体验优化**
  - 优化左侧菜单滚动条视觉样式：更细、更圆润、低干扰、悬停增强

- 🔖 **版本信息一致性**
  - 关于页展示版本号已同步更新为 `v0.4.2`

## 技术说明

- 当前 CI 会在 `main` 分支 push 后构建并推送 Docker `latest` 镜像。
- 本次更新暂未单独发布带版本号的镜像 tag（如 `:0.4.2`），默认通过 `:latest` 获取。

## 兼容性与注意事项

- 语义召回能力依赖你在设置页启用并正确配置：
  - Base URL
  - API Key
  - Embedding Model
- 若 Embedding 服务不可用，助手会自动降级为普通分析，不阻断主流程。

## 致谢

感谢大家对国际化与助手能力的反馈，这一版主要聚焦“可用性 + 能力落地”，后续会继续补齐更多页面英文覆盖与检索可观测能力（索引状态、重建索引等）。
