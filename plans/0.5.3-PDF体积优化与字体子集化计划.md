# 0.5.3 - PDF 体积优化与字体子集化计划

> 目标：在保留中文 PDF 导出能力的前提下，继续降低字体资源对包体积、加载链路和缓存压力的影响。

## 1. 背景

当前 0.5.2 已完成：
- 中文 PDF 导出能力
- 中文字体按需加载，避免静态进入主入口

但仍存在：
- 字体文件本身体积较大
- 首次导出 PDF 时仍需加载完整字体资源
- PWA / 构建产物总体积受到影响

因此需要补一个更小范围的后续计划，聚焦“字体文件本身的体积控制”。
