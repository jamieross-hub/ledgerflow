## 3. 执行原则

1. 优先最小改动，不大改导航体系
2. 若能复用现有 `FinancePage`/资讯页结构，则优先复用
3. 页面命名要和内容一致，避免“点资讯进工具”的错位感
4. 每完成一项后同步更新计划状态

---

## 4. 验收标准

1. 工资工具拥有独立入口或独立承载页
2. 市场资讯页只保留资讯相关内容
3. 导航名称、页面标题、页面内容三者一致
4. 至少通过 `npm run build` 验收
