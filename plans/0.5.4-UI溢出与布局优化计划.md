# 0.5.4 - UI 溢出与布局优化计划

> 目标：修复当前界面中常见的横向溢出、卡片内容撑破、表单/表格在窄屏下挤压错位等问题，优先处理影响主流程的页面。

## 1. 背景

当前 0.5.3 已完成字体体积优化与 Docker 交付补齐，但前端界面仍存在一些零散的 UI 溢出问题，包括：
- 卡片/表格在窄屏下横向撑出
- 长文本、长按钮、长标签未正确换行
- flex / grid 子项缺少 `min-width: 0` 导致内容挤爆
- 局部面板在移动端的滚动与留白不稳定

本计划聚焦“最小代价收口 UI 溢出问题”，不做大规模视觉重设计。

1. 核心页面在常见窄屏宽度下无明显横向溢出

3. 构建通过
4. 仅做必要样式优化，不做大规模 UI 重构
