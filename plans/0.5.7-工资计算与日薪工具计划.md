# 0.5.7 - 工资计算与日薪工具计划

> 目标：在不做大改版的前提下，为 LedgerFlow 增加一组实用、低学习成本的工资计算工具，优先覆盖“月薪倒推日薪 / 时薪”“工作日工资估算”“加班工资估算”等高频场景。

## 1. 范围

本轮优先放到现有 `FinancePage` 内，先做一个轻量工具区，避免额外新增导航复杂度。

## 3. 验收标准

1. 用户可直接输入月薪并看到日薪 / 时薪结果
2. 输入非法值时有明确提示，不出现 `NaN` 或空白结果污染页面
3. 加班工资至少支持 3 类倍数场景
4. 改动尽量控制在 `FinancePage` 与必要的工具函数范围内
5. 至少通过 `npm run build` 验收
